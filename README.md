# 英単語暗記ゲーム - スマートフォンアクセス手順

## 方法1: Pythonの簡易サーバーを使用（推奨）

### 手順

1. **コマンドプロンプトまたはPowerShellを開く**

2. **ゲームフォルダに移動**
   ```
   cd C:\Users\ryuya\word-game
   ```

3. **サーバーを起動**
   ```
   python -m http.server 8000
   ```

4. **PCのIPアドレスを確認**
   
   別のコマンドプロンプトで以下を実行:
   ```
   ipconfig
   ```
   
   「IPv4 アドレス」を探します（例: `192.168.1.100`）

5. **スマートフォンからアクセス**
   
   スマートフォンのブラウザで以下のURLを開く:
   ```
   http://[PCのIPアドレス]:8000/index.html
   ```
   
   例:
   ```
   http://192.168.1.100:8000/index.html
   ```

### 注意事項
- PCとスマートフォンが**同じWi-Fiネットワーク**に接続されている必要があります
- ファイアウォールでポート8000が許可されている必要があります

---

## 方法2: Node.jsのhttp-serverを使用

### 手順

1. **http-serverをインストール**（初回のみ）
   ```
   npm install -g http-server
   ```

2. **ゲームフォルダに移動**
   ```
   cd C:\Users\ryuya\word-game
   ```

3. **サーバーを起動**
   ```
   http-server -p 8000
   ```

4. **表示されたURLをスマートフォンで開く**
   
   コンソールに表示されるネットワークアドレス（例: `http://192.168.1.100:8000`）をスマートフォンのブラウザで開く

---

## 方法3: オンラインホスティング（誰でもアクセス可能）

### GitHub Pagesを使用

1. **GitHubにリポジトリを作成**

2. **ファイルをアップロード**
   - index.html
   - style.css
   - game.js
   - words.json

3. **GitHub Pagesを有効化**
   - Settings → Pages → Source: main branch

4. **公開URLをシェア**
   ```
   https://[ユーザー名].github.io/[リポジトリ名]/
   ```

---

## トラブルシューティング

### サーバーが起動しない
- Pythonがインストールされているか確認: `python --version`
- ポート8000が既に使用されている場合は別のポート（8080など）を使用

### スマートフォンからアクセスできない
- PCとスマホが同じWi-Fiに接続されているか確認
- ファイアウォール設定を確認
- PCのIPアドレスが正しいか確認

### 単語データが読み込めない
- words.jsonファイルが同じフォルダにあるか確認
- ブラウザのコンソール（F12）でエラーを確認
